# RE:COLLAR — Battle Demo

タイミング判定で戦う、ブラウザ動作の 2D RPG 戦闘デモ。
剣士 **エリシア** が仲間とともに、**過去の幻影（2体）→ 暗黒騎士アデル → 魔王アリア** の
3 ステージを駆け抜けます。

> 大学の講義で作成した作品「**To The End**」を前身とし、その世界観・キャラクター・スキルを
> リメイク／拡張したものです。
> **Phaser 3 / Vanilla JavaScript **

## ⚔ ゲームの核：タイミングバトル

このデモの肝は「**攻撃も防御もタイミング次第**」という点です。

### プレイヤーターン
4 つのコマンドから選択します。

- **こうげき** … 3 連撃。流れるバーに合わせて 3 回タイミング入力。
  3 回すべて **PERFECT** で決めると **PERFECT COMBO** フィニッシュ（特大ダメージ＋演出）。
- **スキル** … MP を消費して放つ特殊技（サブメニューから選択）。原作「To The End」の技を踏襲：
  - **ヘヴィースラッシュ**（−12MP）… 高威力の一撃
  - **ホーリー・レイン**（−14MP）… ダメージ＋**HP回復**（聖魔法）
  - **ロード・オブ・ナイト**（−12MP）… ダメージ＋**HP吸収**（与ダメージの30%回復）
- **ぼうぎょ** … 構えて MP を回復。次の敵攻撃のダメージを半減し、ガード判定も広くなる。
- **ひっさつ（OVERDRIVE）** … **OD ゲージ**が満タンのときだけ発動可能な必殺技。
  スロー演出からの光刃斬りで超ダメージ＋画面全体の演出。ここぞの一手。

### OD（オーバードライブ）ゲージ
PERFECT / GOOD 判定を決めるたびに左下の **OD ゲージ**が溜まります。
満タンになると `READY!` が点灯し、**ひっさつ**が解禁。連戦をまたいで持ち越せるので、
ボス戦に合わせて温存する戦略も可能です。

### 敵ターン（ここも遊べる）
敵の攻撃は一方的に受けるだけではありません。振りかぶりに合わせて **ガード／パリィ** のタイミング入力が発生します。

- **PERFECT（パリィ成功）** … ダメージほぼ無効化＋MP回復＋火花演出
- **GOOD（ガード）** … ダメージ半減
- **MISS（直撃）** … フルダメージ＋画面シェイク

ボスの **魔王アリア** は専用の大技を持ちます：
- **[ダークスラッシュ]** … 渾身の一撃（パリィ必須）
- **[ダークヒール]** … HPを回復してくる（押し切られる前に削り切れ！）

### 判定とコンボ
- `PERFECT` / `GOOD` / `MISS` の 3 段階判定。
- `PERFECT` を連続させると **COMBO** が伸び、リザルトの **RANK（S〜C）** に反映されます。

## 👥 登場人物・ストーリー（原作「To The End」より）

- **エリシア** … 本作の主人公。剣士。世界を救うため、旧王都の王城を目指す。
- **暗黒騎士アデル** … 魔王に仕え、行く手を阻む堕ちた騎士。
- **魔王アリア** … かつてエリシアと理想を共にした仲間。裏切られ、復讐の果てに魔王となった。
  その結末は、勝利してなお苦い ―― 「かつての友を、討ち果たした…」。

> 原作のステージ進行（過去の幻影 → 暗黒騎士アデル → 魔王アリア）を踏襲し、
> **3 ステージの連戦**として再構成しています。1 面の過去の幻影は **2 体**（撃破すると 2 体目が出現）。

### 🤝 仲間（バックライン・サポート）
エリシアは一人では戦いません。前線には出ず、**後方から自動でサポート／追撃**する仲間がいます。
（※キャラ画像は未実装のため、現在はプレースホルダー表示）

- **シズ（銃使い）**
  - **追撃射撃** … PERFECT COMBO を決めると、確率で追撃の一射を放つ
  - **パッシブ：狙撃支援** … PERFECT 判定のダメージが常時 +20%
- **ノエル（支援・治癒）**
  - **パッシブ：再生の祈り** … 毎ターン HP を少し回復
  - **守護** … 被弾時、確率でダメージを軽減

## 💻 ローカルで動かす

Phaser は画像・音声を `XMLHttpRequest` で読み込むため、ブラウザの CORS 制約により
**`index.html` のダブルクリック（`file://`）では動作しません。** 簡易 HTTP サーバ経由で開いてください。

```bash
# このフォルダで実行（どちらか）
npx serve            # ← 推奨（大きめの音声ファイルも安定して配信）
python -m http.server 8000
# → ブラウザで表示された http://localhost:**** を開く
```

> 環境によっては `python -m http.server` が大きいファイル（BGM 等）の配信を
> 途中で打ち切ることがあります。読み込みが止まる場合は `npx serve` を使ってください。
> （ゲーム側にも読み込みタイムアウト＋自動リトライを実装済みです）
